# 面向对象

字面量 —— 字面上显示的东西叫做字面量

​	var a = 1;

包装类 —— 没有new的函数声明就是包装类

​	var a = Number(1);



什么是对象：

1.只要是对象就可以有自己的私有属性

2.只要是new出来的都是对象

3.不同对象肯定不会相等

4.对象都会有引用机制



this

1.在普通函数下，this永远指向window

2.有事件源则指向事件源本身

3.在定时器下 除了es6 的this，this执行window

4.在对象下this指向自己本身



不是所有的东西都是绝对的

但window是绝对的

只需要控制类的prototype，就可以控制类的所有方法
