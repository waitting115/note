# Java笔记

## 1.历史背景

创始人：James · Gosling   高斯林  高司令

博士毕业后进入IBM，不被重视  后跳槽到SUN（Stanford University Network）

开发智能家居的时候，当时最火的C++有很多弊端，索性就自己开发一套语言来用，最初起名为Oak（橡树），研发出来后又不被硬件商重视。

1995年，Oak -->  Java    SUN在美国硅谷IT盛会中正式发布

名字来源：印度尼西亚有个爪哇岛盛产咖啡豆 (猫屎咖啡  麝香猫)

1996年发布JDK1.0

1997.2发布JDK1.1

1998.12 发布JavaEE

1999.6发布Java第二代 JDK1.2

J2SE   (Java 2 Standard Edition)  Java第二代标准版   做桌面应用  C/S

J2EE  (Java 2 Enterprise Edition)  Java第二代企业版  做web应用  B/S

J2ME  (Java 2 Micro Edition)  Java第二代微型版   开发移动端

2004.5  发布JDK 1.5，加了很多新特性 Java 5

2005.6  发布 Java第六版  是最经典的版本

Oracle公司在2009面以74亿$并购SUN公司

Oracle公司创始人：拉里 · 艾里森

2011  Java7

2014  Java8

2017  Java9

Java特点：跨平台、面向对象、简单性、健壮性（鲁棒性）、多线程、大数据开发。

## 2.Java的跨平台性

JVM虚拟机  -->  Java Vitual Machine

是Java的翻译官

工作机制：在内存中开辟一块空间，将java源文件编译为字节码文件。

源文件：.java   字节码文件：.class

JRE   Java Runtime Environmeent   java的字节码文件运行环境

JRE包含JVM

JDK  java development kit   开发工具包

JDK包含JRE

JDK中：

​	bin  全都是工具（javac.exe   java.exe等）

​	include  包含了其他语言写的程序

​	jre  运行环境

​	lib   写好的java类

​	src.zip   java 的源代码
