# java二级考试刷题笔记

1.程序流程图中带有箭头的线段表示：控制流。

2.结构化程序设计的基本原则包括：自顶向下，模块化，逐步求精。

3.软件设计中模块化分应遵循的准则是：高内聚低耦合。

4.在软件开发中，需求分析阶段产生的文档是：软件需求规格说明书。（可行性分析阶段产生可行性分析报告；总体设计阶段产生概要设计说明书；概要设计阶段产生集成测试计划。

5.算法的有穷性是指：算法程序的运行时间是有限的。

6.除了堆排序算法的比较次数是O(nlog2n)，快速排序、冒泡排序、直接插入排序等算法的比较次数都是n(n-1)/2。

7.栈：先进后出，插入和删除都是在栈顶操作。

8.在数据库设计中，将E-R图转换成关系数据模型的过程属于：逻辑设计阶段。

9.由关系R和S通过运算得到关系T：笛卡尔积是用R集合中的元素为第一元素，S集合中的元素为第二元素构成的有序对。

10.java语言标识符命名规则为：类名和接口名每个单词的首字母大写；变量和方法名遵循驼峰命名法；常量名所有字母全部大写。

11.java独立于平台，主要得益于虚拟机，只有字节码与各个操作系统和硬件分开才能做到独立于平台。

12.Boolean类型值不能与其他类型值相互转换。

13.Java中+有拼接功能，str + num ==> strnum。

14.char类型是个特殊的类型，每个字符都有一个十进制数与之对应，当char类型相加减时，就是十进制的数相加减得到的结果。 

15.在Java中，子类继承父类，子类功能要比父类强大，能在其基础上增加自己的属性和方法，另外也能重写父类中的方法，但无法更换和减少父类的方法和属性。

16.除数不能为0异常：ArithmeticException.

17.在Java中，所有异常的父类都为Exception类，自定义异常继承Exception类。

18.append()方法属于stringbuffer和stringbuild中的方法，不属于string。

19.所有字节输入输出流类都继承自：InputStream类和OutputStream类。

20.得到文件下的所有文件名，不能对其内容进行修改只能是File类，FileInputStream，FileOutputStream，RandomAccessFile是对其内容进行操作。

21.getSize()获取窗口大小；getForeground()获取构件前景色；getBackground()获取构件后景色；paint()绘画。

22.Swing构件以J开头。

23.一个标准的线程由线程ID，当前指令指针(PC)，寄存器集合和堆栈组成。线程间数据共享。线程创建除了继承Thread类之外，还可以实现Runnable接口。多线程具有并发性，多线程同时运行，结果可能导致紊乱。

24.Runnable接口中的run()方法没有参数列表，重写时不能添加参数列表，如果添加，编译会出错。

25.同步锁知识点：多个线程调用一个对象的多个方法，这些方法都被synchronized修饰，那么这些线程共同竞争一把锁，最后表现的就是同步顺序执行各个被synchronized修饰的方法。

26.Applet获取参数是通过HTML文件中采用<PARAM>标记定义的参数。



# 第二套

1.循环队列有队头和队尾两个指针，但循环队列仍是线性结构。在循环队列中需要队头和队尾两个指针来共同反映队列中元素的动态变化。

2.当有序线性表为顺序存储时才能用二分查找。可以证明的是对于长度为n的有序线性表，在最坏的情况下二分查找只需比较log2n次，而顺序查找需比较n次。

3.链式存储结构即可以针对线性结构也可以针对非线性结构。链式存储结构中每个节点都是由数据域与指针域组成，增加了存储空间。顺序存储结构的存储一定是连续的，链式存储结构的存储空间不一定是连续的。

4.数据流图中带箭头的线表示的是数据流，即沿着箭头方向传送的数据通道，一般在旁边标注数据流名。

5.在软件开发中，需求分析阶段可以使用的工具书有DFD图、数据字典DD、判定树与判定表。

6.对象有如下一些基本特点：标识唯一性、分类性、多态性、封装性、模块独立性好。

7.数据管理发展至今已经历了三个阶段：人工管理阶段、文件系统阶段、数据库系统阶段。其中最后一个阶段结构简单、使用方便逻辑性强物理性少，在各个方面都表现最好，一直占据数据库里领域的主导地位。

8.java中一个文件只允许一个public类。

9.java特点：多线程、单继承、跨平台、动态性。只有接口可以多继承。

10.八进制前面加0，十六进制前面加0x。

11.在JDK1.7之前，switch不支持string类型，会报错。

13.类变量是直接定义在类下面的，可以直接通过类名去调用，用static修饰成静态变量才能实现类名调用。不用static修饰的变量叫做实例变量，不用static修饰只能通过对象的引用去调用。

14.下标越界异常：ArrayIndexOutOfBoundsException

15.捕获并处理异常：try-catch     直接抛出不处理：throw

# 第三套

1.循环队列是线性结构的线性表。

2.栈支持子程序调用。在主程序调用子函数时要首先保存主程序的执行状态，然后转去执行子程序，将子程序的执行结果放到主程序中调用子程序的位置，继续向下执行，这种调用符合栈的特点。

3.二叉树的基本性质3：在任意一棵二叉树中，度为0的节点总比度为2的节点多一个，所以有5个度为2 的节点，这个树的总节点即为5+1=6。

4.冒泡排序、简单选择排序和直接插入排序在最坏的情况下比较的次数都是n(n-1)/2次，而堆排序在最坏的情况下比较的次数是nlog2n次。

5.   软件测试的目的只是找到错误，并不涉及改正。

​	程序调试的基本步骤：错误定位，修改设计和代码，以排除错误，进行回归测试，防止引进新错误。

​	程序调试通常也称DeBug。

​	软件测试的基本准则有：所有测试都应追溯到需求、严格执行测试计划，排除测试的随意性，充分注意测试中的群及现象，程序员应该避免测试自己的程序，穷举测试不可能，妥善保管测试计划等文件。

6.模块独立性：每个模块只完成自己的子功能，并且与其他模块联系最少且接口简单。优秀的软件应做到  高内聚低耦合，耦合性是模块之间互相连接的紧密程度的度量 ，而内聚性是一个模块内部各个元素之间彼此结合的紧密程度。

7.数据库应用系统中的核心问题是  数据库的设计。

8.java所使用的的是16位的Unicode码。

9.在java中具有精度的只有浮点数，float为单精度，double为双精度。默认浮点数为double类型，如果指定单精度，需在数值后面加F或f。

10.逻辑运算符：&& || !   位运算符：>> << >>> <<< 

11.除数为0报错：java.lang.ArithmeticException

12.java.io：

​		StringBufferInputStream
